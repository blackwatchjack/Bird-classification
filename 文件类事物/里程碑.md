# 项目里程碑 (Milestones)

## 阶段 1：后端数据引擎与 API 开发 `(完成度 40%)`
- 任务 1.1：完善 IOCDataLoader，将 Excel 数据转化为适配 el-tree 的嵌套 JSON 结构。
- 任务 1.2：实现 FastAPI 接口，暴露 /scan（开始扫描）、/tree（获取分类树）和 /photos/{species_id}（获取特定物种照片列表）接口。
- 任务 1.3：集成极速扫描器，确保后端能在 1 分钟内处理 10 万量级的文件名匹配。

## 阶段 2：Element Plus 前端骨架搭建
- 任务 2.1：初始化 Vue 3 项目，配置 Element Plus 全局引入。
- 任务 2.2：实现响应式布局，左侧使用 el-scrollbar 包裹 el-tree-v2，中央区域建立响应式栅格网格。
- 任务 2.3：对接 Mock 数据，完成分类树的折叠、搜索和筛选功能逻辑。

## 阶段 3：前后端联调与功能闭环
- 任务 3.1：对接真实扫描接口，实现扫描进度条实时反馈（使用 WebSocket 或长轮询）。
- 任务 3.2：实现“地址指引”功能，确保点击 Web 界面按钮能触发本地资源管理器打开。
- 任务 3.3：缩略图优化，由后端动态生成 Web 可读的缩略图流，避免直接读取 GB 级原图。

## 阶段 4：优化与交付 (Electron 封装)
- 任务 4.1：使用 Electron 将 Web 页面打包，处理本地文件协议缓存问题。
- 任务 4.2：压力测试，验证 10 万张照片下 el-tree-v2 的渲染性能与内存占用。
